# SignUpSystem

用python写的一个基础型二维码签到系统，配合vue
上传人员名单使用excel文件

安装介绍：

1.安装python3
2.获取签到系统压缩包
3.将压缩包的SignUp文件夹解压到桌面
4.解压完成之后打开cmd
5.使用以下命令(输入一次使用一次回车)：
cd Desktop
cd SignUp
pip install -r requirements.txt
python main.py
6.出现网页地址，将其输入至浏览器完成搭建

使用介绍：

IP以实际ip为准
账户：admin
密码：admin
登录：http://IP:8000/home#/Login
签到：http://IP:8000/home#/sign
